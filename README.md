# 🚦 Bus Traffic Prediction System

## 📌 Project Overview

This project focuses on predicting traffic volume (number of vehicles) using time series data. The system analyzes historical traffic patterns and uses a machine learning model to estimate future traffic conditions. This can be extended to predict bus arrival times in real-world scenarios.


## 🎯 Objectives

* Analyze traffic data using time-based features
* Build a machine learning model for prediction
* Evaluate model performance
* Deploy the model using Streamlit for real-time prediction

## 📊 Dataset

* Source: Traffic dataset (CSV)
* Features:

  * DateTime
  * Junction
  * Vehicles (target variable)
  * ID


## ⚙️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Streamlit


## 🔧 Steps Involved

1. Data Collection
2. Data Preprocessing
3. Feature Engineering (hour, day, month, day_of_week)
4. Model Building (Random Forest Regressor)
5. Model Evaluation (MAE, RMSE)
6. Deployment using Streamlit
